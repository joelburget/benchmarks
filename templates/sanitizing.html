Sanitizing

What We Do

Before we store any text on our server, we sanitize it to prevent attacks. We remove any Javascript or HTML that could potentially be used for an XSS attack.

How?

We use html5lib to sanitize the input. We prefer using a well tested library to writing our own. You can see all the tags that html5lib will allow at their repository. Note that we do not allow mathml or svg.
