Getting Started

Get a username

This guide assumes you have a username. If you don't have one yet please email Professor Bruce Weide at weide (dot) 1 (at) osu (dot) edu to set up an account. Please include your name, organization (if any), and why you would like to participate

Post

Once you have your account set up the next step is to post something. Click the link in the toolbar at the top which will take you to an editor. Simple html tags such as <p>, <b>, and <code> are allowed. Javascript is not. Files can be included through hyperlinks or by using the file upload button below the text field. Posts can be classified as Problem, Realization, Verification, or Other. See other posts on the site for clarification of the distinction.

Comments

Comments are included for every post. The same html tags that can be used in posts can be used in comments.
